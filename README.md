# Git - Beginners Guide

## Steps to configure git


```
git config --global user.name "Pranav Jain"
```

```
git config --global user.email pranav@abc.com
```


Set carriage return

Windows:

git config --global core.autocrfl true

Linux:
```
git config --global core.autocrfl input
```


Edit config file
```
git config --global -e
```

Help
```
git config --help
```
